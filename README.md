# TwiTrends
TwiTrends is a java tool to stream the lists of the top 50 subjects on Twitter for 62 countries.

Requirements:

Twitter API authentification in /resources/config.properties: 
- ConsumerKey
- ConsumerSecret
- AccessToken
- AccessTokenSecret

-------------------------------------------------------------------------------------------------

How it works?

java -jar TwiTrends.jar
